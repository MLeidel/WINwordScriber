Word Scriber

(Windows Version)

Python 3.x must be installed. https://www.python.org/downloads/
Then install the necessary library that is used by Word Scriber:

C:\wsr>pip install pywebview

If and when I make a Windows install for this application, I will include the pywebview install as part of it.

Before using wsr.pyw open the options.dat file with a text editor (default: notepad) and review the initial settings. In particular make sure the text editor and file manager are items installed on your system. In Windows the file manager is called explorer.

Run the App:

c:\wsr\python wsr.pyw

If and when I make a Windows install for this application, it will put an icon on your desktop.

Often you will need to select the text that you want to apply formatting to. Note that indent and outdent do not have buttons. Tab is indent, and Shift-Tab is outdent.

Hot Keys

Ctrl-s ... Quick Save
Ctrl-q ... Quit
Ctrl-e ... Edit Options.dat file
Ctrl-z ... Undo
Ctrl-t ... Redo
Ctrl-f ... Launch File Manager
Ctrl-i ... Italic
Ctrl-u ... Underline
Ctrl-b ... Bold
Tab    ... Indent
Shift-Tab  Outdent

Once Word Scriber is installed it will not require an Internet connection.

File and Graphics

Files can be opened and stored anywhere. The default location for files in in the application directory in a folder called "docs". Graphic file must be stored in or under the application directory. An "images" folder is provided by the installation. There is a button that will open your systems file manager to the applications directory.

HTML

If or when you open the HTML editor (using the HTML button) you will notice that the HTML seems to be, shall we say, "messy" and not efficiently written. This is because the internal code (not my code) that produces the HTML supporting the visible document generated the HTML code in this manner. If you try to make it better, or "clean it up", it will just change it back as long as the document is brought into this application. There is some debate as to whether the Javascript that drives this process will ever be enhanced, but for now, it does work.

Options file

The options.dat file in the application's directory contains settings important to initialization of the app on startup. 
  • font
    • like Arial or DejaVu Serif
  • font size
    • like 11pt
  • font color
    • #000 or black
  • text line height
    • like 125%
  • background color of app
    • like #abcdef or lightblue
  • background color of editing area
    • like #fff or white
  • App window WIDTH on startup
    • a whole integer, like 700
    • represents pixels
  • App window HEIGHT on startup
  • text editor
    • name of a text editor installed on your system
    • Its only use at this time is to open the options.dat file
    • like gedit or mousepad or sublime or ...
  • file manager
    • name of a file manager install on your system
    • It will help with document and graphic management
    • like thunar or caja or nautalis or ...
